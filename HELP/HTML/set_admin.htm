Настройки администратора

 Вверх  Назад  Вперед

Программа не в состоянии полноценно работать без настройки и поодержании в актуальном состоянии:

шаблонов документов

базы данных экспертиз

базы данных нормативно-правовых актов и их комментариев

общих переменных (заполнить каждую переменную и ее возможные значения)

 

Доступ ко многим настройкам администратора производится из Главное меню - Сервис - Настройки - Администратор.

 

 

Перед проведением настроек рекомендуется создать резервную копию всех файлов программы.
