Синхронизация с OUTLOOK

 Вверх  Назад  Вперед

Синхронизация с Microsoft Outlook может осуществляться:

по запросу (Главное меню - Дела - Контроль в Outlook)

в автоматеческом режиме (Главное меню - Сервис - Контроль сроков в Outlook) - по умолчанию отключена

 

 

Автоматический режим синхронизации может незначительно замедлять время открытия и закрытия дела.

 

События в Outlook задаются в виде встреч.

Отключение в Outlook оповещения встречи влечет ее автоматическое удаление при синхронизации.
