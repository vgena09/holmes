Переменные дела

 Вверх  Назад  Вперед

Переменные дела используются только для ОДНОГО уголовного дела.

В настоящее время такая переменная только одна - номер уголовного дела.

 

Окно настройки переменных дела вызывается:

автоматически - при создании нового уголовного дела;

вручную - через Главное меню - Дела - Переменные.

 

 

Как правило, значение данных переменных используется в качестве значения по умолчанию для переменных документа.

Поэтому, задав их однажды, не нужно будет впоследствии тратить время на выполнение многих рутинных действий.
