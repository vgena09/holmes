Переменные документа

 Вверх  Назад  Вперед

Переменные дела используются только для ОДНОГО документа.

В зависимости от их значения формируется текст документа.

Они могут просто подставляться в текст документа, могут определять наличие определенных блоков текста (например при применении фото-, видеофиксации и т.д.), могут влиять на контрольные сроки и т.д. (все зависит от алгоритма, заложенного разработчиком шаблона документа).

 

Окно настройки переменных документа вызывается:

автоматически - при создании нового документа;

вручную (документ должен быть не завершен - то есть выделен красным цветом) - через Главное меню - Документы - Открыть или командой открыть в меню секции документов.
