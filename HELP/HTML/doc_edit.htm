Редактировать документ

 Вверх  Назад  Вперед

Выделите документ и дважды кликните по нему мышкой или нажмите Ввод.

 

 

Если документ не готов, то будет предложено создать его (задать переменные, сформировать).

Для измененияпризнака готовности документа - выделить его и нажать пробел.

 

Если нужно пересоздать документ (на основе уже заданных переменных), то помечаем его неготовым и даем команду на его открытие.
